Uwolnij moc skryptowania z HomeyScript. Automatyzuj wszystko, o czym kiedykolwiek marzyłeś, nawet jeśli Flows nie są dla Ciebie wystarczające!

HomeyScript to oparty na JavaScript język skryptowy, który współpracuje z Homey Web API i różnymi funkcjami Homey Apps SDK.

Między innymi, możesz:

— Kontrolować i monitorować urządzenia Homey.
— Uruchomić HomeyScript z Flow, nawet z Tagami.
— Sprawić, że Flow będzie kontynuowane tylko wtedy, gdy HomeyScript powie 'True!'
— Tworzyć i zarządzać Tagami Flow z HomeyScript.
— Uzyskać dostęp do API dowolnej strony internetowej z HomeyScript.
— Sprawić, by Homey mówił dynamiczny tekst.
— I wiele, wiele więcej...

Po instalacji, odwiedź https://my.homey.app/script na swoim Macu lub PC, aby rozpocząć skryptowanie.
